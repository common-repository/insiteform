Track and analyse your WordPress forms to improve conversions and user experience.

== Description ==
The Insiteform plugin works with [Insiteform](https://www.insiteform.com), a powerful tool for WordPress users who want to track and improve the performance of their forms. With its analytics and tracking features, you can increase your form's conversion rate and boost your business.

== Installation ==
1. Download the plugin files
2. Navigate to the "Plugins" section of your WordPress dashboard and click "Add New"
3. Click "Upload Plugin" and select the downloaded zip file
4. Click "Install Now" and activate the plugin
5. Once the plugin is activated, you will see a new menu item titled "Insiteform" in the dashboard sidebar
6. Set up your [Insiteform](https://www.insiteform.com) account
7. Copy your generated activation key
8. Click on the "Insiteform" menu item and enter your activation key
9. Click "Save Key" to activate the plugin
